## Divergence

There is no symmetric line to add the `@sinclair/typebox/compiler` import to. Otto's `extension-load-perf.test.ts` already resolves loader.js from `packages/pi-coding-agent/dist/...`, not the gsd-pi `packages/dist` path; the test change is a no-op (or worse, a regression to the wrong path) in otto. The `search-loop-guard.test.ts` jiti-avoidance edit is gsd-pi-specific test scaffolding for theme deps that otto doesn't pull in the same way.

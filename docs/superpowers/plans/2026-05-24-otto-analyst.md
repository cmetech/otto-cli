# OTTO Analyst Capability Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Ship the MVP "analyst" capability — OTTO can ingest a local data file, analyze it with SQL, and produce a saved interactive dashboard, triggered by plain language, resumable across sessions.

**Architecture:** One bundled extension at `src/resources/extensions/analyst/`. A `DuckDbSession` wraps an on-disk DuckDB database co-located with the OTTO session. Ingest loads local files into DuckDB. A `ScratchpadRuntime` (main-thread `LocalRuntime` in this MVP) runs SQL cells; long cells dispatch through an `AsyncJobManager` so the user keeps working. Deliverables are typed output folders under `~/.otto/deliverables/<slug>/` with provenance, rendered to a self-contained ECharts dashboard.

**Tech Stack:** TypeScript, `@duckdb/node-api` (native), `@sinclair/typebox` (tool + metadata schemas), `node:test` + `node:assert/strict`, OTTO `ExtensionAPI` / `ToolDefinition`, reused `AsyncJobManager`.

---

## Planning decisions (refining the spec's open items)

- **Runtime backend (spec §11.1):** MVP uses a **main-thread `LocalRuntime`** executing **SQL cells**. DuckDB's `@duckdb/node-api` runs queries on native threads and returns promises, so the JS event loop stays responsive; turn-level non-blocking comes from `AsyncJobManager`. The **`worker_thread` backend + TS cells + embedded `getLlm()/sample()` helpers** are the first follow-up increment behind the same `ScratchpadRuntime` interface (not in this plan).
- **DuckDB distribution (spec §11.2):** native node addon (`@duckdb/node-api`).
- **Charting (spec §11.3):** ECharts, inlined into a self-contained HTML page.
- **Background threshold (spec §11.4):** cells with `estMs > 5000` (or `background: true`) dispatch as jobs.
- **Metadata validation:** use `@sinclair/typebox` + `@sinclair/typebox/value` (already a dependency) instead of adding zod.
- **Job tracking:** the analyst extension instantiates its own `AsyncJobManager` (the class is exported from the async-jobs extension) and extends the shared `JobType` union with `"analysis"`. Unifying `/jobs` across extensions is out of scope; analyst long-runs surface via completion follow-ups.

## Security model (LLM-generated SQL)

The scratchpad runs **SQL generated by an LLM from prompts** — and ingested data can carry prompt-injection that steers later queries. DuckDB is **not** a sandbox by default: raw SQL can read/write arbitrary files (`read_csv('/etc/passwd')`, `COPY … TO`), reach the network (`INSTALL httpfs`), load native extensions, and `ATTACH` other databases. The plan defends at the **engine level**, not with fragile regex blocklists:

1. **Locked-down session instance.** The DuckDB instance that runs LLM cells is created with `enable_external_access=false`, `allow_community_extensions=false`, `allow_unsigned_extensions=false`, `lock_configuration=true` (so SQL can't re-enable them), plus `memory_limit`/`threads`/`max_temp_directory_size` to bound DoS. This neutralizes file/network/extension/attach at the engine — no blocklist to bypass.
2. **Isolated privileged ingest.** Because `enable_external_access` is instance-global and fixed at creation, file reading is done by a **separate, short-lived in-memory instance** (external access ON) that runs **only our fixed reader** (`read_csv_auto`/`read_parquet`/`read_json_auto`) on a **validated path** — never LLM SQL. Rows are pulled into JS and bulk-loaded into the locked session instance via a tested, type-aware literal builder. The session DB never touches the filesystem.
3. **Path policy (both controls).** `classifyIngestPath()` rejects remote-looking paths (`://`) and non-files; **allows** files inside an allowed root (cwd + configured data dirs) without friction; and **requires user confirmation** (`ctx.ui.confirm`) for files outside the allowed roots. Both layers apply.
4. **Timeout + interrupt.** Cells honor an `AbortSignal` and a hard per-cell timeout via `connection.interrupt()`, so a runaway query is killable (inline) or cancellable (`cancel_job`, background).
5. **Defense-in-depth.** A lightweight pre-check rejects `INSTALL/LOAD/ATTACH/COPY/EXPORT/PRAGMA/SET` in cells — mostly redundant once the engine is locked, but yields clearer errors.

**Alignment with Anton:** Anton runs arbitrary Python in a venv with no capability sandbox (relying on env isolation + an optional OS firewall + credential vault). For the SQL MVP, OTTO is **stronger** on capability lockdown. Two deferred parity items: the **credential vault** (`DS_*` env injection, never in code) when auth-gated ingest lands, and **worker-level isolation** when TS cells arrive (which re-introduce Anton's arbitrary-code threat model).

```ts
// src/resources/extensions/analyst/duckdb-session.ts
import { DuckDBInstance } from "@duckdb/node-api";
import type { DuckConfig } from "./secure-config.ts";

type Row = Record<string, unknown>;
export interface ColumnDef { name: string; type: string }

export class DuckDbSession {
	private constructor(
		private readonly instance: Awaited<ReturnType<typeof DuckDBInstance.create>>,
		private readonly connection: Awaited<ReturnType<Awaited<ReturnType<typeof DuckDBInstance.create>>["connect"]>>,
		readonly path: string,
	) {}

	static async open(path: string, config: DuckConfig = {}): Promise<DuckDbSession> {
		const instance = await DuckDBInstance.create(path, config);
		const connection = await instance.connect();
		return new DuckDbSession(instance, connection, path);
	}

	async exec(sql: string): Promise<void> {
		await this.connection.run(sql);
	}

	async query(sql: string): Promise<Row[]> {
		const reader = await this.connection.runAndReadAll(sql);
		return reader.getRowObjects() as Row[];
	}

	/** Best-effort interrupt of the running query (used by timeout/abort). */
	interrupt(): void {
		(this.connection as { interrupt?: () => void }).interrupt?.();
	}

	/** Create a table and bulk-insert JS rows. No filesystem access required —
	 *  this is how ingest moves data into the locked session instance. */
	async loadRows(table: string, columns: ColumnDef[], rows: Row[]): Promise<void> {
		const colDefs = columns.map((c) => `${quoteIdent(c.name)} ${c.type}`).join(", ");
		await this.exec(`CREATE OR REPLACE TABLE ${quoteIdent(table)} (${colDefs})`);
		if (rows.length === 0) return;
		const names = columns.map((c) => c.name);
		const BATCH = 500;
		for (let i = 0; i < rows.length; i += BATCH) {
			const chunk = rows.slice(i, i + BATCH);
			const values = chunk
				.map((r) => `(${names.map((n) => sqlLiteral(r[n])).join(", ")})`)
				.join(", ");
			await this.exec(`INSERT INTO ${quoteIdent(table)} VALUES ${values}`);
		}
	}

	async close(): Promise<void> {
		this.connection.closeSync();
		this.instance.closeSync();
	}
}

/** Quote a SQL identifier; doubles embedded double-quotes. */
export function quoteIdent(name: string): string {
	return `"${String(name).replace(/"/g, '""')}"`;
}

/** Serialize a JS value as a safe SQL literal (used only for our own ingested data). */
export function sqlLiteral(v: unknown): string {
	if (v === null || v === undefined) return "NULL";
	if (typeof v === "number") return Number.isFinite(v) ? String(v) : "NULL";
	if (typeof v === "bigint") return v.toString();
	if (typeof v === "boolean") return v ? "TRUE" : "FALSE";
	if (v instanceof Date) return `'${v.toISOString()}'`;
	return `'${String(v).replace(/'/g, "''")}'`;
}
```

```ts
// src/resources/extensions/analyst/path-policy.ts
import { resolve, sep } from "node:path";
import { existsSync, statSync } from "node:fs";

export type IngestDecision = "allow" | "confirm" | "reject";
export interface PathClassification { decision: IngestDecision; reason: string; resolved: string }

function isWithin(child: string, root: string): boolean {
	const r = resolve(root);
	const c = resolve(child);
	return c === r || c.startsWith(r.endsWith(sep) ? r : r + sep);
}

/** Classify an ingest path: reject remote/non-files, allow inside roots, else confirm. */
export function classifyIngestPath(rawPath: string, allowedRoots: string[]): PathClassification {
	if (/:\/\//.test(rawPath)) {
		return { decision: "reject", reason: "Remote paths are not allowed; provide a local file.", resolved: rawPath };
	}
	const resolved = resolve(rawPath);
	if (!existsSync(resolved) || !statSync(resolved).isFile()) {
		return { decision: "reject", reason: "Path is not an existing local file.", resolved };
	}
	if (allowedRoots.some((root) => isWithin(resolved, root))) {
		return { decision: "allow", reason: "Within an allowed root.", resolved };
	}
	return { decision: "confirm", reason: "Outside allowed roots; user confirmation required.", resolved };
}
```

```ts
}

const READERS: Record<string, (p: string) => string> = {
	".csv": (p) => `read_csv_auto('${p}')`,
	".parquet": (p) => `read_parquet('${p}')`,
	".json": (p) => `read_json_auto('${p}')`,
	".ndjson": (p) => `read_json_auto('${p}')`,
};

/** Derive a SQL-safe table name from a file name. */
export function tableNameFor(filePath: string): string {
	const stem = basename(filePath, extname(filePath));
	const safe = stem.replace(/[^a-zA-Z0-9_]/g, "_").replace(/^(\d)/, "_$1");
	return safe || "data";
}

/**
 * Load a local data file into the (locked) session DB.
 *
 * Filesystem reading happens in a SEPARATE, short-lived in-memory privileged
 * instance that runs ONLY our fixed reader — never LLM SQL. Rows are then
 * bulk-loaded into the locked session DB via loadRows (no filesystem access),
 * so the session that runs LLM queries can never read files.
 *
 * NOTE (MVP): the whole file is materialized in JS before load — fine for
 * PM-scale data; a streamed/batched path is a follow-up for very large files.
 */
export async function ingestFile(sessionDb: DuckDbSession, filePath: string): Promise<IngestResult> {
	if (/:\/\//.test(filePath)) throw new Error("Remote paths are not allowed; provide a local file.");
	const ext = extname(filePath).toLowerCase();
	const reader = READERS[ext];
	if (!reader) {
		throw new Error(`Unsupported file type "${ext}". Supported: ${Object.keys(READERS).join(", ")}`);
	}
	const escaped = filePath.replace(/'/g, "''");
	const table = tableNameFor(filePath);

	// --- privileged, isolated read (in-memory; external access ON) ---
	const reading = await DuckDbSession.open(":memory:", permissiveReadConfig());
	let columns: ColumnDef[];
	let rows: Record<string, unknown>[];
	try {
		await reading.exec(`CREATE TABLE staging AS SELECT * FROM ${reader(escaped)}`);
		columns = (await reading.query(
			"SELECT column_name AS name, data_type AS type FROM information_schema.columns WHERE table_name = 'staging'",
		)) as ColumnDef[];
		rows = await reading.query("SELECT * FROM staging");
	} finally {
		await reading.close();
	}

	// --- load into the LOCKED session DB through JS (no filesystem access) ---
	await sessionDb.loadRows(table, columns, rows);
	const sample = await sessionDb.query(`SELECT * FROM ${quoteIdent(table)} LIMIT 20`);
	return { table, rowCount: rows.length, columns, sample };
}
```

```ts
// src/resources/extensions/analyst/runtime.ts
import type { DuckDbSession } from "./duckdb-session.ts";

export interface Cell {
	code: string;
	lang: "sql" | "ts";
	description: string;
	estMs: number;
	background?: boolean;
}

export interface TablePreview {
	columns: string[];
	rows: unknown[][];
	truncated: boolean;
}

export interface CellResult {
	stdout: string;
	tables?: TablePreview[];
	error?: string;
}

/** Pluggable execution backend. MVP ships LocalRuntime; a WorkerRuntime
 *  (worker_thread, TS cells) implements the same interface later. */
export interface ScratchpadRuntime {
	exec(cell: Cell, signal: AbortSignal): Promise<CellResult>;
	view(): Cell[];
}

const PREVIEW_ROW_CAP = 100;
const DEFAULT_TIMEOUT_MS = 30_000;

export class LocalRuntime implements ScratchpadRuntime {
	private readonly cells: Cell[] = [];
	constructor(
		private readonly db: DuckDbSession,
		private readonly timeoutMs: number = DEFAULT_TIMEOUT_MS,
	) {}

	view(): Cell[] {
		return [...this.cells];
	}

	async exec(cell: Cell, signal: AbortSignal): Promise<CellResult> {
		this.cells.push(cell);
		if (cell.lang !== "sql") {
			return { stdout: "", error: "Only SQL cells are supported in the current build." };
		}
		if (signal.aborted) {
			return { stdout: "", error: "Aborted before execution." };
		}

		let timer: ReturnType<typeof setTimeout> | undefined;
		const onAbort = () => this.db.interrupt();
		signal.addEventListener("abort", onAbort, { once: true });

		try {
			const timeoutPromise = new Promise<never>((_, reject) => {
				timer = setTimeout(() => {
					this.db.interrupt(); // best-effort cancel of the running query
					reject(new Error(`Cell timed out after ${this.timeoutMs}ms.`));
				}, this.timeoutMs);
			});
			const rows = await Promise.race([this.db.query(cell.code), timeoutPromise]);

			const columns = rows.length > 0 ? Object.keys(rows[0]) : [];
			const capped = rows.slice(0, PREVIEW_ROW_CAP);
			const preview: TablePreview = {
				columns,
				rows: capped.map((r) => columns.map((c) => (r as Record<string, unknown>)[c])),
				truncated: rows.length > PREVIEW_ROW_CAP,
			};
			return { stdout: `${rows.length} row(s)`, tables: [preview] };
		} catch (err) {
			return { stdout: "", error: err instanceof Error ? err.message : String(err) };
		} finally {
			if (timer) clearTimeout(timer);
			signal.removeEventListener("abort", onAbort);
		}
	}
}
```

> Note: `db.interrupt()` is best-effort. If the installed `@duckdb/node-api` does not expose `interrupt`, the timeout still resolves the cell with an error (the abandoned query finishes in the background and is discarded). A hard kill arrives with the deferred `worker_thread` backend.

```ts
// src/resources/extensions/analyst/deliverables/store.ts
import { Type } from "@sinclair/typebox";
import { Value } from "@sinclair/typebox/value";
import {
	mkdirSync, readdirSync, readFileSync, writeFileSync, existsSync, statSync,
} from "node:fs";
import { join } from "node:path";
import { randomUUID } from "node:crypto";

export const DeliverableType = Type.Union([
	Type.Literal("html-app"),
	Type.Literal("document"),
	Type.Literal("dataset"),
]);

const FileEntry = Type.Object({
	path: Type.String(),
	bytes: Type.Number(),
	modifiedAt: Type.String(),
});

const TurnEntry = Type.Object({
	index: Type.Number(),
	timestamp: Type.String(),
	filesTouched: Type.Array(Type.String()),
});

const ProvenanceEntry = Type.Object({
	conversation: Type.String(),
	turns: Type.Array(TurnEntry),
});

export const DeliverableMeta = Type.Object({
	id: Type.String(),
	slug: Type.String(),
	name: Type.String(),
	description: Type.String(),
	type: DeliverableType,
	primary: Type.Optional(Type.String()),
	createdAt: Type.String(),
	updatedAt: Type.String(),
	files: Type.Array(FileEntry),
	provenance: Type.Array(ProvenanceEntry),
});

export type DeliverableMetaT = typeof DeliverableMeta extends infer _ ? import("@sinclair/typebox").Static<typeof DeliverableMeta> : never;

export interface CreateInput {
	name: string;
	description: string;
	type: "html-app" | "document" | "dataset";
	primary?: string;
}

export interface Deliverable { slug: string; path: string; meta: DeliverableMetaT }

export function slugify(name: string): string {
	return name.toLowerCase().trim().replace(/[^a-z0-9]+/g, "-").replace(/^-+|-+$/g, "") || "deliverable";
}

export class DeliverableStore {
	constructor(private readonly root: string) {
		mkdirSync(this.root, { recursive: true });
	}

	private uniqueSlug(base: string): string {
		let slug = base;
		let n = 2;
		while (existsSync(join(this.root, slug))) slug = `${base}-${n++}`;
		return slug;
	}

	async create(input: CreateInput, conversation: string, turnIndex: number): Promise<Deliverable> {
		const slug = this.uniqueSlug(slugify(input.name));
		const path = join(this.root, slug);
		mkdirSync(path, { recursive: true });
		const now = new Date().toISOString();
		const meta = Value.Parse(DeliverableMeta, {
			id: randomUUID().slice(0, 8),
			slug,
			name: input.name,
			description: input.description,
			type: input.type,
			primary: input.primary,
			createdAt: now,
			updatedAt: now,
			files: [],
			provenance: [{ conversation, turns: [{ index: turnIndex, timestamp: now, filesTouched: [] }] }],
		});
		this.write(path, meta);
		return { slug, path, meta };
	}

	async recordTouch(slug: string, conversation: string, turnIndex: number, files: string[]): Promise<void> {
		const path = join(this.root, slug);
		const meta = this.read(path);
		const now = new Date().toISOString();
		meta.files = this.scanFiles(path);
		let prov = meta.provenance.find((p) => p.conversation === conversation);
		if (!prov) { prov = { conversation, turns: [] }; meta.provenance.push(prov); }
		prov.turns.push({ index: turnIndex, timestamp: now, filesTouched: files });
		meta.updatedAt = now;
		this.write(path, meta);
	}

	async list(): Promise<DeliverableMetaT[]> {
		if (!existsSync(this.root)) return [];
		const metas: DeliverableMetaT[] = [];
		for (const entry of readdirSync(this.root, { withFileTypes: true })) {
			if (!entry.isDirectory()) continue;
			const metaPath = join(this.root, entry.name, "metadata.json");
			if (existsSync(metaPath)) metas.push(this.read(join(this.root, entry.name)));
		}
		return metas.sort((a, b) => b.updatedAt.localeCompare(a.updatedAt));
	}

	private scanFiles(path: string) {
		return readdirSync(path)
			.filter((f) => f !== "metadata.json")
			.map((f) => {
				const st = statSync(join(path, f));
				return { path: f, bytes: st.size, modifiedAt: new Date(st.mtimeMs).toISOString() };
			});
	}

	private read(path: string): DeliverableMetaT {
		return Value.Parse(DeliverableMeta, JSON.parse(readFileSync(join(path, "metadata.json"), "utf-8")));
	}

	private write(path: string, meta: DeliverableMetaT): void {
		writeFileSync(join(path, "metadata.json"), JSON.stringify(meta, null, 2));
		writeFileSync(join(path, "README.md"), renderReadme(meta));
	}
}

function renderReadme(meta: DeliverableMetaT): string {
	return [
		`# ${meta.name}`, "",
		meta.description, "",
		`- **Type:** ${meta.type}`,
		`- **Created:** ${meta.createdAt}`,
		`- **Updated:** ${meta.updatedAt}`,
		meta.primary ? `- **Open:** ${meta.primary}` : "",
		"", "## Files", "",
		...meta.files.map((f) => `- ${f.path} (${f.bytes} bytes)`),
	].join("\n");
}
```

// src/resources/extensions/analyst/tools/ingest-tool.ts
import type { ToolDefinition } from "@loop24/pi-coding-agent";
import { Type } from "@sinclair/typebox";
import type { DuckDbSession } from "../duckdb-session.ts";
import { ingestFile } from "../ingest.ts";
import { classifyIngestPath } from "../path-policy.ts";

const schema = Type.Object({
	path: Type.String({ description: "Absolute path to a local data file (.csv, .parquet, .json)." }),
});

type UiCtx = { ui?: { confirm?: (title: string, message: string) => Promise<boolean> } };

export function createIngestTool(
	getDb: () => DuckDbSession,
	getAllowedRoots: () => string[],
): ToolDefinition<typeof schema> {
	return {
		name: "ingest",
		label: "Ingest data file",
		description:
			"Load a local data file (CSV/Parquet/JSON) into the analysis database and return its schema, row count, and a sample. " +
			"Use this whenever the user references a data file they want analyzed or reported on.",
		promptSnippet: "Load a local data file into the analysis database; returns schema + sample.",
		parameters: schema,
		async execute(_id, params, _signal, _onUpdate, ctx) {
			// Path policy: reject remote/non-files; allow in-root; confirm out-of-root.
			const cls = classifyIngestPath(params.path, getAllowedRoots());
			if (cls.decision === "reject") {
				return { content: [{ type: "text", text: `Cannot ingest: ${cls.reason}` }], details: undefined };
			}
			if (cls.decision === "confirm") {
				const ok = await (ctx as UiCtx)?.ui?.confirm?.(
					"Confirm data access",
					`Read a file outside your workspace?\n${cls.resolved}`,
				);
				if (!ok) {
					return {
						content: [{ type: "text", text: `Ingest declined: "${cls.resolved}" is outside the allowed roots.` }],
						details: undefined,
					};
				}
			}
			const result = await ingestFile(getDb(), cls.resolved);
			const colLines = result.columns.map((c) => `- ${c.name}: ${c.type}`).join("\n");
			return {
				content: [{
					type: "text",
					text: [
						`Loaded **${result.table}** (${result.rowCount} rows).`,
						"", "Columns:", colLines,
						"", "Sample (first rows):", "```json",
						JSON.stringify(result.sample.slice(0, 5), null, 2), "```",
					].join("\n"),
				}],
				details: undefined,
			};
		},
	};
}

```ts
// src/resources/extensions/analyst/tools/scratchpad-tool.ts
import type { ToolDefinition } from "@loop24/pi-coding-agent";
import { Type } from "@sinclair/typebox";
import type { ScratchpadRuntime, CellResult } from "../runtime.ts";
import type { AsyncJobManager } from "../../async-jobs/job-manager.ts";

const schema = Type.Object({
	code: Type.String({ description: "SQL to run against the analysis database." }),
	description: Type.String({ description: "One-line description of what this cell does." }),
	estimated_execution_time_seconds: Type.Number({ description: "Estimate; cells over 5s run in the background." }),
});

const BACKGROUND_THRESHOLD_MS = 5000;

function formatResult(res: CellResult): string {
	if (res.error) return `Error: ${res.error}`;
	const t = res.tables?.[0];
	if (!t) return res.stdout || "(no output)";
	const header = t.columns.join(" | ");
	const body = t.rows.slice(0, 20).map((r) => r.map(String).join(" | ")).join("\n");
	const more = t.truncated ? "\n… (preview truncated)" : "";
	return `${res.stdout}\n\n${header}\n${body}${more}`;
}

export function createScratchpadTool(
	getRuntime: () => ScratchpadRuntime,
	getManager: () => AsyncJobManager | null,
): ToolDefinition<typeof schema> {
	return {
		name: "scratchpad",
		label: "Analysis scratchpad",
		description:
			"Run a SQL cell against the analysis database to compute results, aggregate, or shape data for a report. " +
			"Use this whenever the user wants to analyze, summarize, or report on ingested data. " +
			"Long cells (estimate > 5s) run in the background and return a job ID so you can keep working.",
		promptSnippet: "Run a SQL analysis cell against the ingested data.",
		promptGuidelines: [
			"Ingest a file first, then query its table by name.",
			"Prefer SQL aggregation over returning raw rows.",
			"For long scans, set a realistic estimate so the run goes to the background.",
		],
		parameters: schema,
		async execute(_id, params) {
			const runtime = getRuntime();
			const estMs = Math.round(params.estimated_execution_time_seconds * 1000);
			const cell = { code: params.code, lang: "sql" as const, description: params.description, estMs };
			const manager = getManager();

			if (manager && estMs > BACKGROUND_THRESHOLD_MS) {
				const jobId = manager.register("analysis", params.description, async (signal) =>
					formatResult(await runtime.exec(cell, signal)),
				);
				return {
					content: [{ type: "text", text: `Analysis started in the background: **${jobId}**. Use await_job for results, or keep working.` }],
					details: undefined,
				};
			}

			const res = await runtime.exec(cell, new AbortController().signal);
			return { content: [{ type: "text", text: formatResult(res) }], details: undefined };
		},
	};
}
```

```ts
import type { ToolDefinition } from "@loop24/pi-coding-agent";
import { Type } from "@sinclair/typebox";
import type { DeliverableStore } from "../deliverables/store.ts";

const createSchema = Type.Object({
	name: Type.String({ description: "Human-readable deliverable name (the folder slug derives from this)." }),
	description: Type.String({ description: "Short description shown to the user." }),
	type: Type.Union([Type.Literal("html-app"), Type.Literal("document"), Type.Literal("dataset")]),
	primary: Type.Optional(Type.String({ description: "Entry-point filename you will write (e.g. dashboard.html)." })),

export function createDeliverableTools(
	getStore: () => DeliverableStore,
	getTurn: () => TurnRef,
) {

		name: "create_deliverable",
		label: "Create deliverable",
		description:
			"Claim a folder for a user-facing output (dashboard, report, or dataset) and get the absolute path to write files into. " +
			"Call this BEFORE writing the files. The folder lives under the user's deliverables directory and is auto-tracked.",
		promptSnippet: "Claim a folder for a dashboard/report/dataset; returns the path to write into.",
		parameters: createSchema,
		async execute(_id, params) {
			const ref = getTurn();
			const d = await getStore().create(params, ref.conversation, ref.turn);
			return {
				content: [{ type: "text", text: `Created deliverable **${d.slug}**.\nWrite files into: ${d.path}` }],
				details: undefined,
			};
		},
	};

	const listDeliverables: ToolDefinition<typeof listSchema> = {
		name: "list_deliverables",
		label: "List deliverables",
		description: "List all deliverables in the user's deliverables directory (newest first).",
		promptSnippet: "List existing deliverables.",
		parameters: listSchema,
		async execute() {
			const all = await getStore().list();
			if (all.length === 0) return { content: [{ type: "text", text: "No deliverables yet." }], details: undefined };
			const lines = all.map((m) => `- **${m.slug}** (${m.type}) — ${m.name}: ${m.description}`);
			return { content: [{ type: "text", text: ["## Deliverables", ...lines].join("\n") }], details: undefined };
		},
	};

	return { createDeliverable, listDeliverables };
}
```

```ts
// src/resources/extensions/analyst/index.ts
import type { ExtensionAPI } from "@loop24/pi-coding-agent";
import { getDeliverablesDir } from "@loop24/pi-coding-agent";
import { mkdtempSync } from "node:fs";
import { tmpdir } from "node:os";
import { join, dirname, basename } from "node:path";
import { AsyncJobManager } from "../async-jobs/job-manager.js";
import { DuckDbSession } from "./duckdb-session.js";
import { secureConfig } from "./secure-config.js";
import { LocalRuntime, type ScratchpadRuntime } from "./runtime.js";
import { DeliverableStore } from "./deliverables/store.js";
import { createIngestTool } from "./tools/ingest-tool.js";
import { createScratchpadTool } from "./tools/scratchpad-tool.js";
import { createDeliverableTools, type TurnRef } from "./tools/deliverable-tools.js";

export default function Analyst(pi: ExtensionAPI) {
	let db: DuckDbSession | null = null;
	let runtime: ScratchpadRuntime | null = null;
	let manager: AsyncJobManager | null = null;
	let store: DeliverableStore | null = null;
	let conversation = "default";
	let turn = 0;
	let latestCwd = process.cwd();

	const getDb = () => { if (!db) throw new Error("Analyst DB not ready (wait for session_start)."); return db; };
	const getRuntime = () => { if (!runtime) throw new Error("Analyst runtime not ready."); return runtime; };
	const getManager = () => manager;
	const getStore = () => { if (!store) throw new Error("Deliverable store not ready."); return store; };
	const getTurn = (): TurnRef => ({ conversation, turn });
	// Allowed roots for ingest without confirmation: the session workspace.
	const getAllowedRoots = () => [latestCwd];

	function sessionDbPath(ctx: { sessionManager: { getSessionFile?: () => string | undefined } }): string {
		const file = ctx.sessionManager.getSessionFile?.();
		if (file) {
			const dir = file.slice(0, -6); // strip ".jsonl" → per-session dir (see artifact-manager.ts)
			return join(dir, "analysis.duckdb");
		}
		return join(mkdtempSync(join(tmpdir(), "analyst-session-")), "analysis.duckdb");
	}

	pi.on("session_start", async (_event, ctx) => {
		latestCwd = (ctx as { cwd: string }).cwd;
		const path = sessionDbPath(ctx as never);
		const file = (ctx as { sessionManager: { getSessionFile?: () => string | undefined } }).sessionManager.getSessionFile?.();
		conversation = file ? basename(dirname(path)) : "default";
		turn = 0;
		db = await DuckDbSession.open(path, secureConfig()); // LLM cells run locked-down
		runtime = new LocalRuntime(db);
		store = new DeliverableStore(getDeliverablesDir());
		manager = new AsyncJobManager({
			onJobComplete: (job) => {
				if (job.awaited) return;
				const status = job.status === "completed" ? "done" : "error";
				const body = job.status === "completed" ? job.resultText ?? "(no output)" : `Error: ${job.errorText ?? "unknown"}`;
				pi.sendMessage(
					{ customType: "analyst_job_result", content: `**Analysis ${status}: ${job.id}** (${job.label})\n\n${body}`, display: true },
					{ deliverAs: "followUp" },
				);
			},
		});
	});

	pi.on("session_shutdown", async () => {
		if (manager) manager.shutdown();
		if (db) await db.close();
		db = null; runtime = null; manager = null; store = null;
	});

	pi.registerTool(createIngestTool(getDb, getAllowedRoots));
	pi.registerTool(createScratchpadTool(getRuntime, getManager));
	const { createDeliverable, listDeliverables } = createDeliverableTools(getStore, getTurn);
	pi.registerTool(createDeliverable);
	pi.registerTool(listDeliverables);

	pi.registerCommand("deliverables", {
		description: "List analyst deliverables (open / reveal / export)",
		handler: async () => {
			const all = store ? await store.list() : [];
			const content = all.length === 0
				? "No deliverables yet."
				: ["## Deliverables", ...all.map((m) => `- **${m.slug}** (${m.type}) — ${m.name}`)].join("\n");
			pi.sendMessage({ customType: "analyst_deliverables", content, display: true });
		},
	});
}
```
